NAME: LinkUnit

SYNOPSIS: LinkUnit Stream Position Unit Net

DESCRIPTION

This command is used to create a link between single input and output units and external data in the streams. The component number 'Position' of the stream 'Stream' is linked to 'Unit'.

For example, the command:

  • LinkUnit FRUITS 5 banana my_net.rtdnn
    links the unit named "banana" to the 5th component of the stream "FRUITS".

    Usually the command LinkGroup is a more convenient tool for creating these links, but in special cases when it is necessary to link units individually, this command is useful.
