NAME: Move

SYNOPSIS: Move Object Parent NewParent Net

DESCRIPTION

Move is used to move an object of a network from one group to another. Streams, units and sub-groups can be moved. The 'Object' is moved from its current parent, 'Parent', to the new parent 'NewParent' (Note that unlike the analogous unix "mv" command, the current parent must be specified because there is no "working directory").
