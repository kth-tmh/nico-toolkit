NAME: LinkGroup

SYNOPSIS: LinkGroup Stream Group Net

DESCRIPTION

This command is used to create the link between input and output units and the external data in the streams. 'Stream' must have the same number of components as the number of units in 'Group'. The units of 'Group' are linked to the components of 'Stream' in the same order as in the group. If some other ordering is desired or for some other reason units must be linked individually to components of streams, the command LinkUnit should be used (see also the Streams section).
