NAME: Import

SYNOPSIS: Import [options] Object SourceNet Net

DESCRIPTION

Import is used to copy an object in one network ("SourceNet") to another ("Net"). Connections between units within the object are copied automatically. Groups, units and streams can be copied. By default, the new object is made a member of the root-group "network".

OPTIONS

-m Group By default, the new object is made a member of the root-group "network", but this option let's you put the new copy in any group in the network (Can be done with the Move command as well).
