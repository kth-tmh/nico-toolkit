NAME: Join

SYNOPSIS: UnJoin Child Parent Net

DESCRIPTION

Unjoins an object from one of its parents if it is a member of multiple groups. Unjoining an object from its last group is not allowed. Thus, UnJoin can only be used on objects that have been Joined. However, once an object have more than one parent group, it can be unjoined from any of them.

SEE ALSO

Remove, Rename, Join
