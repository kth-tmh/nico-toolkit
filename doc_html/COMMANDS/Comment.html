NAME: Comment

SYNOPSIS: Comment [option] Net

DESCRIPTION

Adds a comment or a warning message to the log book of a network definition file. The log book can be viewed using the -l option of the Display command.

OPTIONS

-w Add a "warning" message instead of a "comment".
