NAME: Remove

SYNOPSIS: Remove [options] Object Net

DESCRIPTION

Remove an object (or a set of connections) from a network. If 'Object' is a unit or a group containing units, all connctions to and from 'Object' are also removed.

OPTIONS

-r Use this option if you want to recursively delete all objects in or under 'Object' (makes sence only if 'Object' is a group). If you try to remove a non-empty group without this option an error message is printed.
-C Dest Use this option to remove connections. All connections from the object 'Object' to 'Dest' are removed.
