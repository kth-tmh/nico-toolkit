NAME: Copy

SYNOPSIS: Copy [options] Object NewName Net

DESCRIPTION

This command is used to make a copy of an object in the network. By default the new copy becomes a member of the root-group "network". Units, groups and streams can be copied. If the object contains units with connections between them, these connections are copied as well as connctions flowing in to the units. Connections flowing ot of the units are also copied by default, but this can be changed by the -c or -s option (this is usually preferred).

OPTIONS

-c The default is to copy all outflowing conncetions of the object. This doubles the net inflow to the units that has connections from the object. This option prevents Copy from making copys of outflowing connections. See also the-s otion for an alternative way of solving this problem.
-s Split outflowing connections. The connection strength of outflowing connections are divided by two for both the original and the new copy.
-m Group The default is to make the new copy a member of the root-group "network", but this option allows you to put the new copy in any group in the network (Can be done with the Move command as well).
