NAME: CreateNet

SYNOPSIS: CreateNet Name FileName

DESCRIPTION

Creates a new network definition file. Except for the root groop "network" and the bias-unit, the network is empty. The argument "Name" is the internal name of the network and the argument "FileName" is the name of the file holding the definition.

SEE ALSO

Addgroup, AddUnit, AddStream, Connect, Import
